### 2.4 评判标准

`code/gsm_answer_extractor.py:extract_answer` 统一七级回退:

1. 截断模型自续的 `Question:` / `Q:`
2. 取 `Final answer` / `####` 标记
3. 取最后一个平衡的 `\boxed{...}`
4. 取最后等式右侧第一个数
5. 取 `So / Therefore / ...` 开头的结论
6. 取 `Answer:` 标记
7. 回退到全文最后一个数

`is_correct(pred, gold)` 用 `fractions.Fraction` 比较,`tolerance=1e-6`。
